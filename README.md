ЗВІТ ПРО ВИКОНАННЯ ЛАБОРАТОРНОЇ РОБОТИ № 06 (ООП)
Виконавець: Лукіян О.В.

Тема: Використання винятків, успадкування та шаблонів проєктування.

ВИКОНАНІ ЗАВДАННЯ
1. Книгарня (Bookstore)
Концепції: Наслідування, Обробка винятків, Інкапсуляція. Опис Реалізації: Створено базовий клас Book та клас-нащадок GoldenEditionBook. GoldenEditionBook успадковує Book та перевизначає властивість Price, збільшуючи ціну на 30%. Реалізовано валідацію автора (прізвище не починається на цифру), заголовка (довжина не менше 3 символів) та ціни (не від'ємна).

2. Людство (Humanity)
Концепції: Наслідування, Інкапсуляція, Поліморфізм. Опис Реалізації: Класи Worker (Працівник) та Student (Студент) успадковують базовий клас Human (Людина). Реалізовано валідацію полів (ім'я/прізвище з великої літери, довжина) та специфічні валідації для Worker та Student. Розраховано заробіток за годину.

3. Онлайн-радіо
Концепції: Спеціалізовані Винятки, Обробка винятків. Опис Реалізації: Створено ієрархію успадкування винятків від базового InvalidSongException. Реалізовано детальні перевірки на ім'я виконавця (3-20 символів), назву пісні (3-30 символів) та тривалість (0-14 хвилин, 0-59 секунд). Використано try-catch для обробки помилок.

4. Mordor
Концепції: Шаблон Фабричний Метод, Ієрархія Класів. Опис Реалізації: Застосовано шаблон Фабричний Метод (FoodFactory) для створення об'єктів їжі та визначення настрою. Клас Gandalf обчислює очки щастя та визначає настрій (Angry, Sad, Happy, JavaScript/Bliss) залежно від суми очок.

РЕЗУЛЬТАТ:
Усі 4 завдання успішно виконано. Лабораторна робота продемонструвала глибоке розуміння ключових принципів ООП, таких як успадкування, поліморфізм, використання спеціалізованих винятків для надійного керування помилками та застосування шаблону проєктування "Фабричний Метод".
